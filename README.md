# Tripleten web_project_around_es
Una aplicación web interactiva para compartir y explorar lugares de Estados Unidos. Este proyecto se enfoca en la manipulación del DOM y la validación de formularios con JavaScript.

## 🛠 Tecnologías
* **HTML5** (Semántico)
* **CSS3** (Metodología BEM y diseño responsivo)
* **JavaScript ES6** (Programación eventos y objetos)

## ✨ Funcionalidades Principales
* **Gestión de Perfil:** Edición de nombre y descripción a través de un popup.
* **Galería Dinámica:**
    * Añadir nuevas tarjetas (título e imagen).
    * Eliminar tarjetas.
    * Dar "Me gusta" (toggle button).
    * Vista previa de imagen en tamaño completo.
* **Validación de Formularios:**
    * Validación en vivo de inputs.
    * Manejo de errores visuales.
    * Desactivación de botones de envío si el formulario es inválido.
* **UX/UI:** Cierre de popups con tecla `Esc` o clic en el overlay.

## 🚀 Cómo usar
1.  Clona el repositorio o descarga los archivos.
2.  Abre el archivo `index.html` en tu navegador web.
